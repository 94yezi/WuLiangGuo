企业介绍

绿环农商

        确山县绿环农业种植专业合作社（简称确山县绿环合作社）位于确山县开发区司法局东50米，合作社与2008年5月10日成立以来，已建立分社24个，拥有社员2800多户，直接服务农田20余万亩，年订单化肥2万余吨，种子400万余斤，农药40万份，同时提供大量农副产品。

主要业务范围

  • 组织社员种植绿色环保无公害农产品，同时为社员采购、供应所需的绿色环保生产资料；
  • 组织收购、销售成员及同类生产经营者生产的产品；
  • 开展成员所需的运输、贮藏、加工、包装等服务；
  • 引进新技术、新品种，开展与绿色农业生产经营相关的技术培训、技术交流和咨询服务等。
